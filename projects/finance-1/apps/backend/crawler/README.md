# 신용카드 상품설명서(PDF) 수집 크롤러

신한카드, KB국민카드, 현대카드의 특정 카드 상품에 대해\
공식 상품설명서 및 약관(PDF)을 자동으로 수집하고, 텍스트와 메타데이터를
추출하는 Python 기반 크롤러입니다.

------------------------------------------------------------------------

## 디렉토리 구조

    apps/backend/crawler/
    ├── config.py           # 대상 카드 리스트 및 매핑 설정
    ├── main.py             # 실행 엔트리포인트
    ├── check_urls.py       # URL 접근 가능 여부 확인 도구
    ├── crawlers/           # 카드사별 크롤링 전략(Strategy Pattern)
    │   ├── base.py         # 공통 Base Crawler
    │   ├── shinhan.py      # 신한카드 크롤러
    │   ├── kb.py           # KB국민카드 크롤러
    │   └── hyundai.py      # 현대카드 크롤러
    └── utils/
        ├── card_matcher.py # PDF 제목과 카드명 매칭 로직
        └── pdf_extractor.py# pdfplumber 기반 텍스트 추출 래퍼

------------------------------------------------------------------------

## uv 기반 실행 방법

본 프로젝트는 빠른 패키지 관리 도구인 **uv**를 사용합니다.

### 1. uv 설치

``` bash
pip install uv
```

### 2. 의존성 설치 (루트에서 1회)

``` bash
uv sync
```

### 3. Playwright 브라우저 설치

``` bash
uv run playwright install chromium
```

### 4. 크롤러 실행

``` bash
uv run python -m apps.backend.crawler.main
```

------------------------------------------------------------------------

## 출력 결과

수집된 데이터는 프로젝트 루트 기준 `datasets/`에 저장됩니다.

-   `datasets/pdfs/`: 다운로드된 PDF 원본 파일\
-   `datasets/text/`: 추출된 텍스트 및 메타데이터(JSON)\
-   `datasets/index.csv`: 전체 문서 수집 결과 인덱스

------------------------------------------------------------------------

## 🛠 설정 방법

수집 대상 카드를 추가하거나 제거하려면\
`apps/backend/crawler/config.py`의 `TARGET_CARDS` 리스트를 수정하면
됩니다.

크롤러는 **엄격한 카드명 매칭(strict matching)** 방식을 사용하여\
지정된 카드만 수집하도록 설계되어 있습니다.
